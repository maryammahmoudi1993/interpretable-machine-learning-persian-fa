# ترجمه فارسی کتاب Interpretable Machine Learning

به پروژه ترجمه فارسی کتاب **Interpretable Machine Learning** خوش آمدید.

این کتاب نوشته‌ی **Christoph Molnar** است و یکی از منابع شناخته‌شده در زمینه‌ی یادگیری ماشین تفسیرپذیر، توضیح‌پذیری مدل‌ها و فهم رفتار مدل‌های جعبه‌سیاه محسوب می‌شود.

## درباره این ترجمه

این پروژه یک ترجمه‌ی غیررسمی و مستقل از کتاب اصلی است.

هدف ترجمه این است که فارسی‌زبانان بتوانند راحت‌تر به مفاهیم مهم حوزه‌های زیر دسترسی داشته باشند:

- یادگیری ماشین تفسیرپذیر
- توضیح‌پذیری مدل‌ها
- مدل‌های جعبه‌سیاه
- روش‌هایی مثل SHAP، LIME، PDP، ICE و ALE
