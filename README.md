## ساخت کتاب به صورت محلی

برای ساخت نسخه HTML کتاب به صورت محلی، مراحل زیر را انجام دهید:

1. نصب [mdBook](https://rust-lang.github.io/mdBook/):
   ```
   cargo install mdbook
   ```
2. اجرای دستور ساخت در ریشه مخزن:
   ```
   mdbook build
   ```
   کتاب در پوشه `book/` ساخته می‌شود.

3. برای مشاهده در مرورگر حین ویرایش:
   ```
   mdbook serve
   ```
   سپس به `http://localhost:3000` مراجعه کنید.
